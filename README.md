# Redirector

A simple browser extension to add user-defined URL redirections.

- WXT https://wxt.dev/
- React https://react.dev/
- Clarity Core https://core.clarity.design/
- Clarity Design System https://clarity.design/

## Development

### Requirements

- pnpm
- node

### Debug

```sh
pnpm install
pnpm dev
```

### Build

```sh
pnpm install
pnpm build
```
